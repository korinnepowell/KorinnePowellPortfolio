My Cats Flask App

A fun Python Flask App

(SDEV300 Building Secure Python Applications - Summer 2024). This is a fun application developed using Python Flask and demonstrates a combination of backend logic and frontend design. The application features a user registration and login system with robust security measures, ensuring that users can access their accounts safely. It also includes detailed logging of user interactions for enhanced security monitoring and audit purposes, which can be used to identify suspicious activity. Upon registering, users are required to create strong passwords, which is verified against a list of common passwords. The system enforces strict password standards, urging users to include a mix of uppercase and lowercase letters, numbers, and special characters. To protect login credentials, passwords are then encrypted and stored securely. Overall, this project is a well-rounded example of modern development.

Road Trip Cost Calculator

Calculating the Cost of a road trip

(CMSC215 - Intermediate Programming - Spring 2024). This project centers around a graphical user interface (GUI) developed using JavaFX to help calculate the costs of a road trip. The application features input fields for various parameters, including distance, gas prices, fuel efficiency (mileage), number of days, hotel costs, food expenses, and additional attraction fees. Users can easily switch between measurement units, choosing either dollars per gallon or per liter, and miles or kilometers.Once the user inputs their data, the interface provides a way to calculate the total cost of their trip based on these inputs. While the design of the GUI is relatively straightforward, the main emphasis is on error handling, accurate calculations, and well-structured code.
